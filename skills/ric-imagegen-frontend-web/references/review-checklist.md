# Web Image Review Checklist

Use this reference before requesting independent visual review.

## Usability

- Correct size, aspect ratio, resolution, and crop safety.
- Clear focal point and usable negative space.
- No watermarks, accidental text, fake marks, or distorted UI.
- Product or subject is recognizable and relevant.
- Asset can be implemented or placed as intended.

## Direction Quality

- Directions differ in hierarchy and concept, not only palette.
- Selected direction matches audience, offer, and brand constraints.
- Visual novelty supports comprehension and action.
- Related assets form a coherent system without identical composition.
- Text strategy is explicit.

## Evidence

- Brief and direction rationale.
- Asset manifest and provenance.
- Contact sheet or comparison set.
- Selected and rejected direction notes.
- Full-resolution selected files.
- Known limitations and implementation guidance.

Send this package to `ric-visual-design-review`. If assets become a coding source, also send matching source metadata to `ric-design-qa`.
