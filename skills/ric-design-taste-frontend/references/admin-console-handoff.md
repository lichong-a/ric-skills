# Admin Console Handoff

`ric-design-taste-frontend` is for landing pages, portfolios, marketing pages, visual brand pages, and non-admin redesigns.

When the request mentions any of these, hand off to `ric-admin-console`:

- 后台管理系统
- 管理后台
- 控制台
- CRUD
- 列表页
- 表格
- 权限
- 角色
- 用户管理
- 运营中台
- 审批流程
- 日志审计
- 数据看板
- 内部工具
- SaaS back office
- admin panel
- management console

Why:

- Marketing-page rules optimize for narrative and visual distinctiveness.
- Admin-console rules optimize for repeated operation, dense information, permissions, table states, query flows, and Chinese enterprise taste.

Expected behavior:

1. Stop applying hero, AIDA, cinematic scroll, portfolio, landing-page bento, and marketing-copy rules.
2. Read `skills/ric-admin-console/SKILL.md`.
3. Use `ric-admin-console/references/*` for shell layouts, page patterns, component contracts, China enterprise visual style, framework adapters, and acceptance checks.
