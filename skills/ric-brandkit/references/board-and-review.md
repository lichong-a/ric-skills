# Brand Boards And Review

Use this reference to define the minimum proof set and prepare independent review evidence.

## Minimum Direction Board

Each direction should show:

- Strategic idea.
- Mark or logo concept without presentation effects.
- Type and color logic.
- Image or pattern language.
- One or two relevant applications.
- Risks and limitations.

Use a layout that supports comparison. Do not fill a fixed grid merely because space exists.

## Application Proof

Choose applications based on actual use:

- Product UI or app icon.
- Website or portal.
- Social or campaign.
- Document, report, packaging, signage, or physical object.
- Light/dark and small-size states.

Random merchandise is not proof unless merchandise is part of the brand's real use.

## Asset Checks

- Inspect mark at small size and single color.
- Inspect light and dark variants.
- Verify wordmark spelling and letterforms.
- Reject watermark, accidental text, fake trademark, or inconsistent construction.
- Record generation provenance and production limitations.

## Independent Review Package

Provide:

- Brand brief.
- Three direction boards for new or substantial rebrand work.
- Selected and rejected rationale.
- Asset manifest.
- Small-size, single-color, light/dark, and application proofs.
- Similarity, cultural, accessibility, and production risks.

Send the package to `ric-visual-design-review`. When applying the system to a real product, send approved source assets and rendered applications to `ric-design-qa`.
