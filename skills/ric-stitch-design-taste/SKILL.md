# RIC Stitch Design Taste

## Role

This is a design-document **modifier**. It records an approved direction; it is not an implicit source of truth, primary executor, or quality gate.

## DESIGN.md Contract

Document only decisions supported by the brief and selected direction:

- product context and intended users
- layout and responsive behavior
- typography, color, spacing, and component language
- asset and motion direction
- accessibility and state requirements
- explicit non-goals and implementation constraints

Do not impose absolute visual rules unrelated to the project. Any example is non-authoritative. The approved solution design and acceptance criteria remain the source of truth.

## Verification

Review the artifact through `ric-solution-design` and the relevant independent review gate before implementation.
